## Лабораторная работа по математической статистике

#### Программа проводит первичную обработку выборки:
- строит интервальный статический ряд;
- полигон частот; 
- полигон относительных частот;
- гистограмму относительных частот;
- график эмпирической функции распределения;
- считает и выводит точечные оценки числовых характеристик.

В результате работы в **output/** сохраняются таблица, 
описывающая статистический интервальный ряд распределения (*table.xlsx*), 
и фотографии четырёх графиков. Также здесь есть пример отчёта, 
в который необходимо ввести полученные данные.

Впоследствии будут добавлены интервальные оценки параметров распределения, 
а также статистическая проверка статистических гипотез.

#### Инструкция
Перед началом работы необходимо внести свои данные: в **input.txt** нужно вставить
выборку своего варианта, в *main_meanings* нужно внести надёжность (*reliability*) и 
уровень значимости (*significance*). 

Затем следует установить библиотеки, необходимые для работы программы: `pip install -r requirements.txt`

После установки библиотек можно запускать код:
+ `python main.py` без отображения графиков
+ `python main.py show-graphs` с отображением графиков
