# Лабораторная работа по математической статистике

# Программа проводит первичную обработку выборки:
- строит интервальный статический ряд;
- полигон частот; 
- полигон относительных частот;
- гистограмму относительных частот;
- график эмпирической функции распределения.

# Программа проводит расчет параметров распределения:
- считает и выводит точечные оценки числовых характеристик;
- считает и выводит интервальные оценки числовых характеристик.

# Программа проводит статистическую проверку статистических гипотез:
- проверка с помощью критерия Пирсона.

В результате работы в **output/** сохраняются таблица, 
описывающая статистический интервальный ряд распределения (*table_1.xlsx*), 
и фотографии четырёх графиков, таблица, отражающая проверку статистической гипотезы
с помощью критерия Пирсона (*table_2.xlsx*). Также здесь есть пример отчёта, 
в который необходимо ввести полученные данные.

Впоследствии будут добавлены проверка с помощью критерия Колмогорова и график теоретической функции.

# Быстрый старт

При первом скачивании репозитория выполнить в терминале:

```shell
copy ".env.example" ".env"; copy "report.docx" "report_Name_Lastname.docx"; copy "input.txt.example" "input.txt"
```

Перед началом работы необходимо внести свои данные: в `input.txt` нужно вставить
выборку своего варианта, в `.env` нужно внести надёжность (*reliability*) и 
уровень значимости (*significance*). 

Затем следует установить библиотеки, необходимые для работы программы: 

`pip install -r requirements.txt`

После установки библиотек можно запускать код:
+ `python main.py` без отображения графиков
+ `python main.py show-graphs` с отображением графиков

Выходные данные будут храниться в `output/`, 
`report_Name_Lastname.docx` переименовать **СТРОГО** в формате `report_{имя}_{фамилия}`
